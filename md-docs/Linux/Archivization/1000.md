## Direct

```bash
zstd -3 -z -T16 -f -k
lz4 -6 -T16 -z -f -k
pigz -6 -p16 -k
```

## Tar

| Typ        | Kompresor | Komenda                                                                 | Output   |
|------------|-----------|-------------------------------------------------------------------------|----------|
| Tar + GZ   | pigz      | tar -cf - file \| pv \| pigz -6 -p16 > X.tgz                            | X.tgz    |
| Tar + ZSTD | zstd      | tar -cf - X/ \| zstd -3 -T16 -o X.tzst                                  | X.tzst   |
| Tar + GZ   | gzip      | tar -cf - x \| pv \| gzip -6 > X.tgz                                   | X.tgz    |
| Tar + BZ2  | bzip2     | tar -cf - x \| pv \| bzip2 -6 > X.tbz2                                 | X.tbz2   |
| Tar + XZ   | xz        | tar -cf - x \| pv \| xz -6 > X.txz                                     | X.txz    |
| Tar + LZ4  | lz4       | tar -cf - x \| pv \| lz4 -6 -T16 > X.tz4                               | X.tz4    |


```bash
tar -czf x.tgz x
tar -cjf x.tbz2 x
tar -cJf x.txz x
tar -cf - x | lz4 -6 -T16 > x.tz4
```
