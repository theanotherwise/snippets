## Direct

| COMPRESSOR | COMMAND |
|------------|---------|
| GZIP       | `tar -czf dir.tgz dir` |
| BZIP2      | `tar -cjf dir.tbz2 dir` |
| XZ         | `tar -cJf dir.txz dir` |
| LZ4        | `tar -cf - dir \| lz4 -6 -T16 > dir.tz4` |
| ZSTD       | `zstd -3 -z -T16 -f -k dir` |
| LZ4        | `lz4 -6 -T16 -z -f -k dir` |
| PIGZ       | `pigz -6 -k dir` |

# Tar

| TYPE | COMMAND |
|------|---------|
| GZ   | `tar -cf - file \| pv \| pigz -6 -p16 > X.tgz` |
| GZ   | `tar -cf - x \| pv \| gzip -6 > X.tgz` |
| ZSTD | `tar -cf - X/ \| zstd -3 -T16 -o X.tzst` |
| BZ2  | `tar -cf - x \| pv \| bzip2 -6 > X.tbz2` |
| XZ   | `tar -cf - x \| pv \| xz -6 > X.txz` |
| LZ4  | `tar -cf - x \| pv \| lz4 -6 -T16 > X.tz4` |


# Fast Directory

tar -cf - us_ids | pv -B 64M | zstd -10 -T0 -f -o 2025-us_ids.tar.zstd
